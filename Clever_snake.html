Запуск игры:

1. Скачайте файл
2. Разархивируйте (желательно используйте 7-zip)
3. Нажмите 2 раза по файлу Clever_snake.exe

Важно!!!

Не вынимайте файл из папки! Он может работать не корректно.

Как обновить

1. Скачайте zip файл с более новой версией
2. По желанию удалите папку со старой версией
Если произошла какая - то ошибка удалите прошлый файл и скачайте новый

Советы:

1. Как удобнее включать приложение
Создайте текстовый документ на рабочем столе с расширением txt
Напишите: cd далее путь к папке Clever_snake (уже разорхивираванной)
На следующе строчке напишите start Clever_snake.exe
Сохраните файл и поменяйте расшерение с txt на bat
Теперь при нажатие на этот файл у вас будет запускаться игра
